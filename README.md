# G.A.I.A.
Quick Description: Gamifying upcycling for Earth; easy-to-follow DIY tutorials

### My Journey
Hey yalls! It's Jovie, back with another cool passion project that just so happened to become my econ assignment, but I still wish to showcase it to you all

#### INTRODUCING G.A.I.A. - A fashion sustainability application that gamifies upcycling by giving simple DIY fashion tutorials

The goal of G.A.I.A is to foster sustainable habits and trendy fashion where DIY becomes the best form of sustainable living
through easy-to-follow tutorials and competition in leaderboards.
